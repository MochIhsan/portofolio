Website Pariwisata Kabupaten Bima

Project ini merupakan rancangan UI/UX sistem informasi pariwisata berbasis website di Kabupaten Bima. Website dirancang untuk membantu wisatawan memperoleh informasi destinasi, event, berita, akomodasi, dokumentasi, rating, ulasan, dan informasi fasilitas secara lebih terpusat.

UI/UX Figma Design Thinking Website

Deskripsi Project

Website Pariwisata Kabupaten Bima dirancang sebagai media informasi terpusat yang menampilkan berbagai informasi wisata. Latar belakang dari project ini adalah masih tersebarnya informasi wisata di berbagai media sehingga pengguna perlu membuka banyak sumber untuk mendapatkan informasi yang lengkap.

Melalui pendekatan Design Thinking, rancangan website dibuat berdasarkan kebutuhan pengguna seperti informasi lokasi, rute, fasilitas, dokumentasi foto/video, rating dan ulasan, event, berita, akomodasi, serta halaman admin untuk pengelolaan data.

Fitur Utama

Informasi Destinasi

Menampilkan daftar destinasi wisata beserta deskripsi, lokasi, rute, dan fasilitas.

Rating dan Ulasan

Pengguna dapat melihat penilaian dan komentar dari pengunjung lain.

Event dan Berita

Website menyediakan informasi event budaya dan berita pariwisata terbaru.

Dashboard Admin

Admin dapat mengelola data destinasi, event, berita, akomodasi, user, dan ulasan.
