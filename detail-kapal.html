Rekapitulasi Pendapatan Kapal

Project ini merupakan sistem berbasis website yang dirancang untuk membantu PT Duta Ramadhani Permai dalam mengelola data pendapatan kapal secara lebih rapi, terstruktur, dan mudah dilaporkan.

Web App Admin Panel Data Analyst Report

Deskripsi Project

Sistem rekapitulasi pendapatan kapal dibuat untuk mempermudah proses pencatatan dan pengelolaan data kapal. Data yang dikelola meliputi nama kapal, jenis muatan, harga sewa, total muatan, pengeluaran, pendapatan, dan laporan.

Dengan adanya sistem ini, proses pencatatan yang sebelumnya dapat dilakukan secara manual menjadi lebih mudah dipantau. Admin dapat menginput data, melihat detail pendapatan, melakukan filter data, dan mencetak laporan sesuai kebutuhan.

Fitur Utama

Input Data Kapal

Admin dapat menginput nama kapal, jenis muatan, dan informasi sewa kapal.

Pengelolaan Pendapatan

Sistem menghitung dan menampilkan data pendapatan berdasarkan data yang dimasukkan.

Filter Data

Pengguna dapat memfilter data berdasarkan kebutuhan laporan.

Cetak Laporan

Laporan pendapatan kapal dapat dicetak untuk keperluan administrasi.
